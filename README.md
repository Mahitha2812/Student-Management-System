# Student Management System
A simple Java console-based Student Management System that allows users to manage student records efficiently.
##Features
-  Add Student
-  View Students
-  Update Student Details
-  Delete Student
-  Search Student
-  Display All Records
## Technologies Used
- Java
- OOP Concepts
- Collections (ArrayList)
## How to Run
1. Clone the repository:
 git clone https://github.com/Mahitha2812/Student-Management-System.git
2. Open in any Java IDE (Eclipse / IntelliJ / VS Code)
3. Compile and run:
   javac StudentmanagementSystem.java
   java StudentmanagementSystem
